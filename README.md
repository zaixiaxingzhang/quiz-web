# 刷题宝 📚

> 一个零依赖、纯单文件的移动端刷题 Web App。无需服务器、无需安装，下载即用。

**v2.3** · SRS 间隔重复学习系统

---

## ✨ 功能一览

### 🎯 学习模式
- **SRS 间隔重复**：基于 SM-2 算法，根据每道题的记忆等级自动安排复习时间，答对间隔变长，答错重置为 10 分钟后再现
- **新题优先**：未做过的题优先出现，保证覆盖全部题目
- 学习时长计时，每分钟自动保存，统计页显示累计学习时长

### 📝 考试模式
- 自定义题数和限时，交卷后统一批阅
- 成绩页显示正确率，支持逐题查看解析

### ❌ 错题专练
- 自动收录所有答错的题，支持单独练习

### 📖 题库管理
- 多文件夹分类，支持**自定义颜色和图标**（汉字 / 字母 / Emoji）
- 点击题目查看 SRS 状态（记忆等级、下次复习时间、难度因子）
- 支持在题目详情添加**个人笔记**，随导出文件保存
- 批量操作：移动、收藏、重置、删除

### 📥 导入方式
| 方式 | 说明 |
|------|------|
| 本地文件 | 从设备选择 `.json` 题库文件 |
| 网络直链 | 输入下载直链（如 GitHub Raw），自动检测格式，15 秒超时 |

### 📤 导出
- Android 原生分享 / 浏览器下载
- 笔记随题库一起导出，SRS 学习进度不导出（本地专属）

### ⚙️ 设置页
- 题库 JSON 格式说明与示例
- AI 一键转换题库提示词（一键复制）
- SRS 算法详解
- 链接导入使用教程
- 技术沟通联系方式

## 🚀 使用方式

### 直接用浏览器打开
1. 下载 [`quiz-app.html`](./quiz-app.html)
2. 手机或电脑浏览器打开即可

## 📋 题库 JSON 格式

```json
{
  "folders": [
    {
      "name": "文件夹名称",
      "questions": [
        {
          "type": "单选",
          "question": "PLC 的全称是什么？",
          "options": ["可编程逻辑控制器", "可编程计算机", "单片机", "继电器"],
          "answer": "A",
          "explanation": "PLC 即 Programmable Logic Controller"
        },
        {
          "type": "多选",
          "question": "PLC 主要编程语言有哪些？",
          "options": ["梯形图", "指令表", "顺序功能图", "流程图"],
          "answer": "ABC"
        },
        {
          "type": "判断",
          "question": "梯形图中线圈不能并联。",
          "answer": "错误"
        },
        {
          "type": "填空",
          "question": "PLC 按结构分为哪两种？",
          "answer": "整体式",
          "answers": ["整体式模块式", "模块式整体式"],
          "explanation": "整体式（一体机）和模块式"
        }
      ]
    }
  ]
}
```

### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `type` | string | ✅ | `单选` / `多选` / `判断` / `填空` |
| `question` | string | ✅ | 题目内容 |
| `options` | array | 选择题必填 | 选项列表，最多 26 个 |
| `answer` | string | ✅ | 单选填 `"A"`，多选填 `"ABC"`，判断填 `"正确"` 或 `"错误"` |
| `answers` | array | 可选 | 填空题多个可接受答案 |
| `explanation` | string | 可选 | 解析内容 |
| `note` | string | 可选 | 个人笔记（答题后添加） |

### 用 AI 快速转换题库

把任意格式的题目发给 ChatGPT / Kimi / 文心一言，加上 App 内「设置 → AI 一键转换题库」里的提示词，AI 会自动输出标准 JSON，直接导入即可。

## 🧠 SRS 算法说明

本 App 使用 **SM-2 间隔重复算法**（Anki 同款），每道题维护三个参数：

| 参数 | 说明 |
|------|------|
| 记忆等级 Level | 从 0 开始，连续答对则升级 |
| 复习间隔 Interval | 下次复习的时间间隔，随等级指数增长 |
| 难度因子 Ease Factor | 初始 2.5，答对增大，答错减小（最低 1.3） |

**间隔规律：**
- 第 1 次答对 → 1 天后
- 第 2 次答对 → 4 天后
- 第 3 次答对 → 约 10 天后
- 答错 / 不会做 → 10 分钟后重新出现，等级归零
