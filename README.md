# 刷题宝

一款单文件 HTML 刷题工具，无需安装，支持浏览器直接使用，也可使用 Android App或直接访问在线网站
https://zaixiaxingzhang.github.io/Quiz-Web/
https://shuatiweb.netlify.app/

---

## 功能概览

### 题型支持
- 单选题、多选题、判断题、填空题
- 填空题可选**精准匹配**（必须完全一致）或**关键词匹配**（允许顺序差异、标点差异、容错一字）
- 支持数学公式（KaTeX）、代码块（highlight.js）、Markdown 渲染

### 题库管理
- 题库以**文件夹**为单位组织，支持增删改查
- 手动新增/编辑题目，支持多个备选答案
- **批量操作**：多选题目后批量删除或移动到其他文件夹
- **导入**：支持 JSON 格式导入，可选合并学习记录或跳过重复
- **导出**：可选文件夹、笔记、SRS 进度、学习统计，通过系统分享面板发送
- **AI 批量转换**：粘贴任意格式题目，AI 自动解析为结构化 JSON

### 学习模式
- **SRS 智能学习**：基于 SM-2 间隔重复算法，自动调度复习时间，优先展示到期题和新题
- **错题优先**：按错误次数权重排序抽题
- **新题优先**：先做从未练习过的题
- **自定义题型**：按题型和数量自由组合，例如"20道单选 + 5道判断"
- **困难模式**：开启后单选题变为填空，需精准输入正确选项的完整内容
- 学习设置自动记忆（题数、模式、自定义配置），下次进入自动恢复
- 文件夹范围默认全选，支持一键全选/全不选
- 答题后显示解析、支持记笔记、可问 AI

### 考试模式
- 随机抽题，可设置题数和限时
- 交卷后显示得分与错题详情
- 同样支持文件夹范围筛选

### 错题练习
- 自动汇集所有答错过的题目
- 支持按文件夹筛选

### AI 功能
- 答题页可对当前题目**问 AI**，获取解析（流式输出）
- AI 回复支持 Markdown、LaTeX、代码高亮
- 支持 `link("关键词")` 工具，AI 可插入可点击的搜索链接
- 回复可一键保存为题目解析
- 多轮对话，对话记录可保存
- 支持配置任意兼容 OpenAI 格式的 API（默认 SiliconFlow）

### 统计与分析
- 首页展示今日待复习、答题数、积分
- **统计页**：累计积分、学习时长、答对/答错总数
- **各文件夹**：已掌握（SRS 等级 ≥ 2）、学习中（答过但未掌握）、未做
- **SRS 算法分析**：难题榜（按难度因子排序）、已掌握榜
- **题库体检**：检测重复题、缺答案、格式异常

### 外观与设置
- 支持浅色/深色模式，设置页可手动切换
- 支持**按时间段自动切换**深色模式（如 22:00 ~ 07:00 自动开启，支持跨夜）
- 全局搜索：支持按题目内容、选项、解析关键词搜索

## 题库 JSON 格式

```json
[
  {
    "type": "单选",
    "question": "题目内容",
    "options": ["选项A", "选项B", "选项C", "选项D"],
    "answer": "A",
    "explanation": "解析（选填）"
  },
  {
    "type": "多选",
    "question": "题目内容",
    "options": ["选项A", "选项B", "选项C"],
    "answer": "AC",
    "explanation": "解析（选填）"
  },
  {
    "type": "判断",
    "question": "题目内容",
    "answer": "正确"
  },
  {
    "type": "填空",
    "question": "题目内容",
    "answer": "主答案",
    "answers": ["主答案", "备选答案1"],
    "fillMode": "exact",
    "explanation": "解析（选填）"
  }
]
```

`fillMode` 可选 `"exact"`（精准匹配）或 `"fuzzy"`（关键词匹配，默认）。

## SRS 算法说明

采用 SM-2 间隔重复算法：

| 状态 | 条件 |
|------|------|
| 未做 | 从未答过 |
| 学习中 | 答过，SRS 等级 < 2 |
| 已掌握 | SRS 等级 ≥ 2（连续答对积累） |

- 答对：等级 +1，复习间隔延长，难度因子小幅上升
- 答错/不会做：等级归零，10 分钟后重新安排，难度因子下降（最低 1.3）
- 答错过的题重新练会后同样可以升入已掌握
