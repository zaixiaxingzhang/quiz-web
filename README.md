# Quiz-Web

# 刷题宝 📚

一个零依赖、纯单文件的移动端刷题 Web App。无需服务器、无需安装、无需构建，下载 `quiz-app.html` 用浏览器打开即可使用，数据存储在本地。

---

## ✨ 主要功能

### 🎯 三种练习模式
| 模式 | 说明 |
|------|------|
| **学习** | 按权重抽题，错题/不会做的题下次优先出现 |
| **考试** | 自定义题数和时间，交卷后统一批阅 |
| **错题** | 专项练习曾经答错且尚未掌握的题目 |

### 📖 题目类型
- 单选题、多选题、判断题、填空题
- 填空题支持多个可接受答案，智能模糊匹配（顺序无关、标点容错）

### 📁 题库管理
- 多文件夹分类，支持**自定义颜色和图标**（汉字 / 字母 / Emoji）
- 批量操作：移动、收藏、重置统计、删除
- 错题本、收藏夹独立入口

### 📥 导入方式
- **本地文件** — 从设备选择 `.json` 题库文件
- **网络链接** — 输入下载直链，自动检测格式（15 秒超时），确认后导入

### 📤 导出
- Android / iOS 原生分享，或直接下载为 `.json` 文件

### 🔍 搜索
- 全局搜索题目内容、选项、解析
- 答题后一键跳百度搜索当前题目

### 🌙 其他
- 深色 / 浅色模式平滑切换
- 积分系统与学习统计
- 支持 Android 实体返回键逐层退回

## 🚀 使用方式

### 浏览器直接打开
1. 下载 [`quiz-app.html`](./index.html)或使用https://zaixiaxingzhang.github.io/Quiz-Web/
2. 用手机或电脑浏览器打开
3. 导入题库或手动新建题目

> 数据存储在浏览器 IndexedDB，建议定期导出备份。

## 📋 题库 JSON 格式

```json
{
  "folders": [
    {
      "name": "文件夹名称",
      "questions": [
        {
          "type": "单选",
          "question": "PLC 的全称是什么？",
          "options": ["可编程逻辑控制器", "可编程计算机", "单片机", "继电器"],
          "answer": "A",
          "explanation": "PLC 即 Programmable Logic Controller"
        },
        {
          "type": "多选",
          "question": "以下哪些是 PLC 的编程语言？",
          "options": ["梯形图", "指令表", "顺序功能图", "流程图"],
          "answer": "ABC"
        },
        {
          "type": "判断",
          "question": "线圈不能并联。",
          "answer": "错误"
        },
        {
          "type": "填空",
          "question": "PLC 按结构分为哪两种？",
          "answer": "整体式",
          "answers": ["整体式模块式", "模块式整体式"]
        }
      ]
    }
  ]
}
```

### 字段说明

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `type` | string | ✅ | `单选` / `多选` / `判断` / `填空` |
| `question` | string | ✅ | 题目内容 |
| `options` | array | 选择题必填 | 选项列表，最多 26 个 |
| `answer` | string | ✅ | 单选填 `"A"`，多选填 `"ABC"`，判断填 `"正确"` 或 `"错误"` |
| `answers` | array | 可选 | 填空题多个可接受答案 |
| `explanation` | string | 可选 | 解析说明 |

## 🛠 技术说明

- 原生 HTML + CSS + JavaScript，**零依赖，零构建**
- 数据持久化：IndexedDB（自动降级到 localStorage）
- 单文件部署，约 **200 KB**
- 兼容 Chrome / Safari / Firefox / Android WebView
